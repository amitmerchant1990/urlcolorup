    jQuery URL ColorUP

    The plugin allows it to automatically highlight URLs in textareas like those on Twitter.

    Pull requests are always welcomed if you find any bugs or you want to make it better.

    Usage

    $(document).ready(function() {
   $("textarea").urlcolorup();
});
