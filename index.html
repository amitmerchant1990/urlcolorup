jQuery URL ColorUP

This jQuery plugin allows it to automatically highlight URLs in textareas like those on Twitter.

How to use jQuery URLcolorUP

If you want to use this jQuery plugin, simply add those lines to the <head> part of your website:

<script src="js/jquery-1.10.2.min.js" type="text/javascript"></script>
<script src="js/jquery.autosize.js" type="text/javascript"></script>
<script src="js/jquery.urlcolorup.js" type="text/javascript"></script>

Warning: Do not include js/jquery-1.10.2.min.js if you already included jQuery to your website!

After including the neccessary files, you can simply add jQuery #hashtags to an existing by using this code (inside an appropriate <script>-element):

$(document).ready(function() {
   $("textarea").urlcolorup();
});
